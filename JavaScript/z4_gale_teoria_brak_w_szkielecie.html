2) dlaczego jest to język skryptowy
JavaScript jest językiem skryptowym, bo jest interpretowany przez przeglądarkę i nie musi być wcześniej kompilowany do kodu maszynowego. Wystarczy przeglądarka, żeby zobaczyć efekty jego działania

3) gdzie jest wykonywany JS (klient lub serwer)
...JavaScript jest wykonywany po stronie klienta (w przeglądarce), ale może też działać na serwerze...

4) dlaczego jest JS to bezpieczne narzędzie
JavaScript jest bezpieczny, bo ma ograniczony dostęp do zasobów komputera i działa w przeglądarce, chroniąc system przed nieautoryzowanymi operacjami.

5) czy jest to język obiektowy, jeśli tak to dlaczego
JavaScript jest językiem obiektowym, bo używa obiektów, metod i zdarzeń do opisu właściwości i działania elementów. Możesz tworzyć własne obiekty i korzystać z gotowych.

8) Do aplikacji mobilnych można użyć Framework
Tak, do aplikacji mobilnych można użyć frameworków takich jak React Native, które pozwalają tworzyć aplikacje na różne systemy operacyjne

9) Aplikacje responsywne to:
Aplikacje responsywne to takie, które automatycznie dostosowują wygląd i układ do rozmiaru ekranu urządzenia, np. telefonu, tabletu czy komputera.
